§l/§redititem - Opens a menu to edit the held item.
§l/§rdfgive <item> - Gives you a specified item.
§l/§ritemdata - Shows the held item's (client-side) NBT data.
§l/§rtemplates - Shows recently used Code Templates.
§l/§runpack - Extracts the contents in the held item (such as items in chests)
§l/§rwebview - Views the held Code Template in your web browser.
§l/§rsendtemplate - Sends the held Code Template to connected 
third-party software.
§l/§rrelativeloc <targer> <f> <u> <r> <rd> <rr> - Gives you a 
Relative Location item.
§l/§rcount - A useless command to waste time!

§l/§rnbs player - Gives you the Song Player functions Code Template.
§l/§rnbs load <filename> - Imports a Note Block Song (.nbs) to a Code Template
To use the load command, place .nbs files in .minecraft/CodeUtilities/NBS Files§f.

§l/§rcolor rgb <r> <g> <b> - Creates a hex color code based on RGB values.
§l/§rcolor hex <hex-code> - Creates a hex color code based on hex values.
§l/§rcolor hsb <h> <s> <b> - Creates a hex color code based on HSB values.
§l/§rcolors - Shows a simple hex color picker in the chat.
§l/§rgradient <startColor> <endColor> <text> - Shows a text colored in gradient.
§l/§rheads - Opens a menu to search and get custom heads.
§l/§rnode <1|2|3|4|5|beta> - Sends you to the specified DiamondFire node.
§l/§ruuid <username> - Grabs the Text item with UUID of the specified player.
§l/§rplots - Opens the plots gui without using the item.
